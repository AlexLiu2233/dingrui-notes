# dingrui-notes — public preview site

This repo is the **public** GitHub Pages site for Dingrui Scholars notes
(**notes.dingruischolars.com**). It contains **only the free preview tier** —
the first 3 units of each course (130 pages) — plus locked teaser cards that
route to the signup funnel.

> **Generated, not hand-edited.** This tree is built from the private source
> repo (`dingrui-products`) by selecting only `tier: "free"` pages from
> `rag/freemium_manifest.json`. The full study guides, practice sets, solutions,
> and PDFs live in the **private** source repo and are served to enrolled
> students through the backend gate — they are intentionally **not** in this
> public repo.
